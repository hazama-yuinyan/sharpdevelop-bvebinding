BVE5 Language Binding for SharpDevelop beta release

2013/3/22
(C) HAZAMA


BVE5 Language Binding for SharpDevelopアドインをダウンロードしていただき、ありがとうございます。このアドインはSharpDevelop上でBVE5のルート定義ファイルに対してシンタックス・ハイライティングやコードコンプリーション、その他ルート作成時に役立つと思われるツールを追加するものです。

--インストール方法--
すでにSharpDevelopをインストールしている方は、同じフォルダーにあるBVE5Binding.sdaddinというファイルをダブルクリックすれば、自動的にSharpDevelopのアドインマネージャーが起動してアドインをインストールできるはずです。SharpDevelopをまだインストールしていないという方は、公式ページから最新版をダウンロードし、インストールしてから上記の手順を踏めば使用できます。

--必須要項--
手元の環境では、残念ながら開発開始時最新版だったSharpDevelop 4.3.0.9338版でしか動作確認をしておりません。なにか不具合等ございましたら、ブログのコメント欄・githubなどでご報告いただけると幸いです。また、追加してほしい機能等のご意見・ご要望等も受け付けております。(当面は当方の考える必要機能の実装、アドインの安定化、独立版エディターのリリース準備などでご要望にお答えするのは難しくなってしまうかもしれませんが、予めご了承ください)

--最後に--
このアドイン制作に手をつけたのが2月中旬頃でした。その頃から構想だけは存在していたのですが、いずれこのアドインが安定化したら、スタンドアローンなエディターとして再リリースしようと思っています。というのも、SharpDevelopではBVE5のルート制作に不必要な機能が沢山付いており、無駄に重くなってしまうため、またSharpDevelop自体の日本語化が公式でサポートされていないためです。個人的にはそれでも一向にかまわないのですが、やはりルート制作者様の役に立ちたいことを考えると、日本語のネイティブサポートは必須事項だと考えています。さらに、そちらの独立版テキストエディターも安定化してきたら、GUIによるCADソフトライクなインターフェイスでのルート作成機能などを実装してBVE5のルート製作用統合開発環境にまで発展させたいという野望もありますが、果たして実現できるのやら・・・
